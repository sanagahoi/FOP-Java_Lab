# Java Pre-Work Exercises

Welcome to the **pre-work** sub-directory! This section contains foundational programming exercises designed to build strong logic, master core syntax, and understand structural implementation before diving into advanced data structures and enterprise architectures.

---

## 📁 Folder Contents

The files and folders in this directory are categorized into core logical segments:

### 1. 🔄 Control Flow & Loop Architectures
Dedicated sub-folders containing deep-dive implementations of Java control structures:
*   `breakStatement/` — Handling loops with explicit break interruptions.
*   `continueStatement/` — Bypassing current loop iterations under specific constraints.
*   `loop/` — Various implementations of `for`, `while`, and `do-while` loops.
*   `switchStatement/` — Multi-directional branch logic execution.

### 2. 🧮 Fundamental Logic Programs
Loose Java classes handling entry-level logic puzzles and operation patterns:
*   `ArithmaticOperations.java` — Basic mathematical operators and processing.
*   `BasicCalculatorUsingSwitchCase.java` — Interactive CLI calculator utilizing switch mechanics.
*   `ContinueAndBreak.java` — Integrated testing of loop controls.
*   `DeclareVariables.java` — Primitive data types and scope initialization.
*   `EvenNoByTernaryOperator.java` — Conditional checking using shorthand ternary logic (`?:`).
*   `FirstProgram.java` — Classic syntax setup and entry points.
*   `ForLoopForOddNumbers.java` — Selective loop iterations focusing on math properties.
*   `ImplementInputOutputOperations.java` — Working with `Scanner` objects to capture input.
*   `LeapYearChecker.java` — Time calculation logic with nested evaluation conditions.
*   `OddOrEvenByIfElseStatement.java` — Standard baseline conditional logic branching.
*   `PalindromeChecker.java` — String/numeric evaluation parsing items backward and forward.
*   `PatternPrinterNestedForLoop.java` — Printing matrix grids and right-angle stars using nested setups.
*   `PrintValuesByWhilelLoop.java` — Conditional execution sequences using indefinitely bound loops.
*   `SwitchCaseStatement.java` — Standard syntax maps for character and integer flags.
*   `ValueInterchanger.java` — Swapping variables using both temporary memory spaces and arithmetic tricks.

## 🛠️ How to Compile and Run Locally

To test any file in this directory using your native terminal, run the following steps:

1. Open your terminal and navigate to this folder:
   ```bash
   cd pre-work
   ```
2. Compile your desired file (e.g., the Palindrome Checker):
   ```bash
   javac PalindromeChecker.java
   ```
3. Run the compiled bytecode file:
   ```bash
   java PalindromeChecker
   ```
